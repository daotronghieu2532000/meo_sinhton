4. API Info

Base URL: https://codego.io.vn/api/

Endpoints:

  • POST /get_token.php - Lấy access token
  • POST /register.php - Đăng ký user
  • POST /login.php - Đăng nhập

💡 Lưu ý: Tất cả API (trừ get_token) đều cần header: Authorization: Bearer {access_token}
